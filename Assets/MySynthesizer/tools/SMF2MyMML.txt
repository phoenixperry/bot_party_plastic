This program convert SMF(standard midi file "*.mid") to MyMML(*.mml.txt).

usage:
> SMF2MyMML input.mid [-v] [-h] [-e 65001] [-p 2] [-o [input.mml.txt]] [-ed]
-v: Verbose
-h: Hit Any Key!
-e: Code page ex.932=sjis,65001=utf8
-p: Port (0~3)
-o: Output to file
-ed: expand drum part (note -> program)
